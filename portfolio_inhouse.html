Raymond James: Brand Standards & Governance Guide

Spec Work — Personal Project

A brand governance document is the most read and least glamorous design artifact in a large organization. It is consulted by vendors who have never met anyone at the company, by regional coordinators making decisions without creative team oversight, and by compliance teams checking materials against regulatory standards. It has to be clear enough that none of those people need to call anyone for clarification.

The 24-page guide covers logo usage rules (minimum sizes, clear space, approved color combinations, prohibited modifications), color system (CMYK, RGB, HEX, and Pantone values for all primary and secondary colors), typography hierarchy (primary, secondary, and fallback fonts with size scales), photography and illustration style guidance, and six pages of side-by-side do/don't examples showing the most common misuse cases pre-empted.

Designed for financial services context with explicit attention to FINRA brand compliance requirements on licensed materials. Deliverables: 24-page PDF (8.5×11 in, InDesign), 1-page color quick reference sheet, 2-page typography specimen, and do/don't examples guide (6 pages). Tools: InDesign, Illustrator, Photoshop.

BayCare Health System: Employee Newsletter Redesign

Spec Work — Personal Project

The internal newsletter is the most-produced, least-considered designed artifact in most large organizations. It gets made on deadline, by whoever has InDesign access, following a template that has not been touched in six years. The result is a document that communicates obligation rather than value — and gets skimmed for three seconds before being set down.

This redesign started with the reader. A clinical nurse reading this on a 15-minute break needs to extract the one or two pieces of information that affect them personally in under two minutes. The hierarchy is built for that reader: the headline communicates the most important story, the four-section grid gives her a visual table of contents, and the HR Update section (statistically the highest-read section in any employee newsletter) gets consistent placement and clear formatting every issue.

The editable InDesign template locks the layout and brand elements while making copy fields easily updateable by the marketing coordinator — no design software expertise required for monthly production. Deliverables: 4-page print newsletter (8.5×11 in, 150 dpi), HTML email version (600 px, Outlook-compatible), digital signage slide (1920×1080 px, PNG), and editable InDesign template (.IDTT). Tools: InDesign, Illustrator, Figma.

Bloomin' Brands: Menu Design Template System

Spec Work — Personal Project

Menu design at a multi-concept restaurant group is a production problem before it is a design problem. There are hundreds of locations, quarterly seasonal updates, regional menu variations, and print vendors in twelve states who all need print-ready files without a designer on call. The design has to be beautiful once and scalable indefinitely.

The system I built separates what is designed (the layout, hierarchy, and visual language) from what is updated (the item names, descriptions, and prices). InDesign paragraph styles lock every font size and spacing decision. Photography slots are fixed dimensions — a new photo either fits the slot or it does not, which eliminates a whole category of layout decisions. The result is a template that a marketing coordinator can update in 45 minutes without touching a single design element.

Deliverables: dine-in menu (11×17 folded to 5.5×8.5 in, CMYK, InDesign), bar menu (8.5×11 in, 4-panel), kids menu (8.5×11 in), to-go insert (5×7 in), and digital menu board zones (1920×1080 px, PowerPoint). Tools: InDesign, Illustrator, Photoshop.

BayCare Health System: Social Media Template Library

Spec Work — Personal Project

Designing a template system is a fundamentally different challenge from designing a single beautiful post. A single post can be perfect. A template system has to be perfect and survive daily use by fifteen different people who did not design it, may not understand brand guidelines, and are working under deadline pressure every week.

The fifteen Figma templates in this system are locked at the frame level — colors, fonts, spacing, and layout are fixed. Only the labeled text layers and designated image slots can be edited. This sounds restrictive, but it is the restriction that makes the system valuable: a marketing coordinator at Morton Plant Hospital can produce a compliant, on-brand recruitment ad in under ten minutes without reviewing the brand guidelines. The template is the guide.

Deliverables: 15 Figma templates across 5 content categories (health tips, service announcements, recruitment, events, stories), all sized for their respective platforms (1080×1080, 1200×628, 1080×1920 px). Includes a 2-page Figma usage guide for coordinators. Tools: Figma, Illustrator.

Raymond James: 20-Size Display Ad Template Library

Spec Work — Personal Project

Every digital campaign at a large financial services firm requires the same creative in 20 different sizes. If the design team approaches this as 20 separate design problems, they spend two days per campaign on mechanical resizing instead of on strategy and concept. If they build it as a component system first, they spend two days once and deploy it in minutes forever after.

The Figma component library I built for this project uses a single master design frame as the source of truth. All brand decisions — colors, typefaces, spacing, logo placement, and CTA button treatment — live in shared styles that update every size simultaneously when changed. The copy fields are the only editable elements in each size variant. Campaigns are customized by changing three text fields across all 20 sizes in under five minutes.

Deliverables: 20-size Figma component library covering all IAB standard sizes (300×250, 728×90, 160×600, 300×600, 970×250, 320×50, 320×480, and 12 additional standard formats), plus LinkedIn sponsored content (1200×627 px). All sizes exported as PNG and documented for HTML5 animation handoff. Tools: Figma, Illustrator, Adobe Animate.
